# Weather Supreme

A feature-rich weather application built with Material 3 Expressive design, supporting forecast, observations, nowcasting, air quality, pollen, and alerts from multiple weather sources.

## About

Weather Supreme is a free and open source weather app that provides:
- Daily and hourly forecasts
- Real-time weather alerts
- Air quality monitoring
- Pollen tracking
- Beautiful visualizations with Material 3 design

## Features

### Weather Data
- Daily and hourly forecasts up to 16 days
- Precipitation in the next hour
- Severe weather and precipitation alerts
- Temperature, feels like temperature, and normals
- Wind speed and direction
- Air quality monitoring
- Pollen & Mold tracking
- Humidity, UV index, visibility, and pressure
- Sun and moon information

### User Interface
- Material 3 Expressive design
- Detailed 24-hour charts
- Automatic dark mode
- Customizable widgets
- Live wallpaper support

### Privacy & Accessibility
- No personal data collection
- No trackers or automatic crash reporters
- Optional location services
- Screen reader support
- Multiple language support

## Technology Stack

- **Built with:** Kotlin
- **UI Framework:** Jetpack Compose
- **Design:** Material 3 Expressive
- **Minimum API:** Android 5.0 (API 21+)
